# Myskin

#### myskin cosmetics shopping app admin panel

## Installation

```bash
1. git clone https://github.com/MyskinxMavka/myskinAdmin.git

   or download a zip file

   https://github.com/MyskinxMavka/myskinAdmin/archive/refs/heads/main.zip

2. In project folder.
   npm install or yarn
```

## Usage

run project

```bash
npm run start
```

###### or

```bash
yarn start
```
